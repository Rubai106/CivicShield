# Sprint-1 Server (Features 1-5)

## Included features
1. Report creation with title and description
2. Category selection
3. Anonymous reporting (authority view masking)
4. Evidence upload and listing
5. Location text + coordinates

## Run
1. Copy `.env.example` to `.env` and set values
2. Install dependencies: `npm install`
3. Run migration: `npm run migrate`
4. Start API: `npm run dev`
